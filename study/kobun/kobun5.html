古文練習　第5問

『徒然草』より（高校入試レベル）

本文

【徒然草】 「花は盛りに、月は隈なきをのみ、見るものかは。 雨に向かひて月を恋ひ、垂れこめて春の行方知らぬも、 なほ、あはれに情け深し。」 （問題用に一部を抜粋・編集）

問題

問1
傍線部「なほ、あはれに情け深し。」を現代語訳しなさい。
問2
「花は盛りに、月は隈なきをのみ、見るものかは。」とありますが、作者が言いたいこととして最も適当なものを選びなさい。
  1. 花も月も、最も美しい時しか見る価値はない。
  2. 花や月を見る必要はない。
  3. 花や月は、最も美しい時だけでなく趣のある姿にも価値がある。
  4. 花より月の方が美しい。
問3
作者はどのような美しさを大切にしていると考えられますか。40字以内で説明しなさい。
問4
本文中の「あはれ」の意味として最も適当なものを選びなさい。
  1. 悲しい
  2. 趣がある・しみじみとした味わいがある
  3. 楽しい
  4. 驚いた
問5
本文の内容として最も適当なものを選びなさい。
  1. 作者は満開の花しか認めていない。
  2. 作者は曇りの日を嫌っている。
  3. 作者は完全ではないものにも趣を感じている。
  4. 作者は月より花が好きだと述べている。

解答・解説

問1
やはり、しみじみと趣があり、心を打つ。

問2
C

問3
最も美しい状態だけでなく、不完全な姿や見えないものにも趣を感じる美しさ。

問4
B

問5
C

解説
『徒然草』では、「満開の花」「満月」のような完成された美だけではなく、 散り際の花や雲に隠れた月などにも深い趣があると述べています。 重要古語
  • なほ … やはり・それでも
  • あはれ … しみじみとした趣・情趣
  • 隈なし … 陰りがない・完全な
