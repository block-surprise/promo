古文練習　第6問

『宇治拾遺物語』の世界観をもとにした入試風問題

本文

　昔、ある僧が山寺へ向かふ途中、一人の老人に出会ひけり。 　老人は重き荷を負ひながら坂を上りゐたり。僧は急ぎの用ありけれど、その姿を見て荷を持つ手助けをしたり。 　老人は何度も礼を述べ、 「人は情けある心こそ、何よりの宝なれ。」 と言ひける。 　僧はその言葉を聞き、「学問や読経ばかりに心を向け、人を助くることを忘れてゐたり。」と深く恥ぢけり。 　寺へ着きて後も、その日の出来事を忘れず、人に親切にすることを心がけける。やがて人々は、その僧を「徳ある僧」と呼ぶやうになりけり。

問題

問1
傍線部「情けある心」を現代語で説明しなさい。
問2
僧が老人を手助けした理由として最も適当なものを選びなさい。
  1. 老人に命令されたから。
  2. 老人が困っている様子を見たから。
  3. 荷物をもらいたかったから。
  4. 寺へ行く道を知りたかったから。
問3
「深く恥ぢけり」とあるのはなぜですか。35字以内で答えなさい。
問4
老人の言葉から読み取れる考えとして最も適当なものを選びなさい。
  1. 学問だけが人を立派にする。
  2. 財産が人にとって最も大切である。
  3. 思いやりの心は何よりも価値がある。
  4. 年を取ると人は弱くなる。
問5
本文の内容として最も適当なものを選びなさい。
  1. 僧は老人を助けず寺へ急いだ。
  2. 老人は寺の住職だった。
  3. 僧は老人との出会いをきっかけに人への思いやりを大切にするようになった。
  4. 老人は僧に学問を教えた。

解答・解説

問1
思いやりや人を大切にする心。

問2
B

問3
学問ばかりを重んじ、人を助けることの大切さを忘れていたと気付いたから。

問4
C

問5
C

解説
この問題は『宇治拾遺物語』によく見られる「人との出会いを通して大切なことに気付く」というテーマをもとにした入試風問題です。 重要古語
  • けり … ～た（過去）・～だなあ（詠嘆）
  • なれ … ～である
  • 恥づ … 恥ずかしく思う
  • 情け … 思いやり・慈悲の心
